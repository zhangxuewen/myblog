今天用了一点的时间把博客从jekyll迁移到了hexo，主要是因为感觉hexo的生态更好，而且写作起来更舒服，方便。

* 环境准备

```javascript

    安装Git、Node.js以及npm工具

    brew install git

    bew install node

    #查看是否安装成功

    node -v

    npm -v
```

* 安装Hexo并创建文件

```javascript
    npm install hexo -g

    hexo init <folder>  #folder为你创建博客的文件夹名称

    cd <folder>
```
